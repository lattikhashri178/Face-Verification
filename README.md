# 🔍 Face Verification System

A web application that verifies if two face images belong to the same person using AI and deep learning.

## 📌 Project Goal
To build a face verification system that can accurately compare two face images and determine if they belong to the same person.

## 🛠️ Technologies Used
- Python
- Flask (Web Framework)
- face_recognition library
- OpenCV
- HTML, CSS, JavaScript

## ✨ Features
- Upload two face images
- AI based face detection
- Face matching with accuracy
- Simple and clean web interface
- Fast verification result

## 📅 Project Timeline
- **Week 0** – Installed tools and libraries, set up environment
- **Week 1** – Face detection using OpenCV
- **Week 2** – Face encoding and comparison logic
- **Week 3** – Full system built and tested

## ▶️ How to Run
1. Install requirements:
   pip install flask face_recognition opencv-python
2. Run the app:
   python app.py
3. Open browser:
   http://127.0.0.1:5000
